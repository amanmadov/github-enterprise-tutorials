# Pluralsight GitHub Courses

## Overview

This document curates comprehensive learning paths and courses focused on GitHub and DevOps practices. These skill paths are designed to help developers, administrators, and security professionals master essential tools and methodologies for modern software development. Whether you're looking to enhance your GitHub administration capabilities, automate workflows with GitHub Actions, implement advanced security measures, or integrate DevOps practices with Azure, these courses provide structured learning experiences with hands-on exercises and real-world scenarios.

## `GitHub Administration` Skill Path

- Platform: Pluralsight
- Course Instructor: Matt Allford and others
- Link: https://app.pluralsight.com/paths/skills/github-administration
- About: The GitHub Administration path is designed for IT professionals managing GitHub at an enterprise level. This path covers essential aspects of GitHub Enterprise deployment, security, user management, and platform maintenance. You'll learn enterprise-grade administration skills including identity management, access control, compliance enforcement, and enterprise-wide policy implementation. The curriculum emphasizes practical scenarios and best practices for maintaining a secure, efficient, and scalable GitHub environment across your organization.
- Courses:
    - **Users and Identity Management**
        - GitHub Administration: The Big Picture
        - Supporting GitHub Enterprise for Users and Stakeholders
        - Managing Identities and Authentication in GitHub
    - **Licensing, Permissions, and Actions**
        - GitHub Deployment and Licensing
        - Managing Access and Permissions in GitHub
        - Enabling Secure Software Development in GitHub
        - Managing GitHub Actions
        - Managing GitHub Packages


## `GitHub Actions` Skill Path

- Platform: Pluralsight
- Course Instructor: Multiple instructors
- Link: https://app.pluralsight.com/paths/skills/github-actions
- About: The GitHub Actions path is designed to help you master workflow automation and continuous integration/continuous deployment (CI/CD) within the GitHub ecosystem. This learning path prepares you for the GitHub Actions certification exam by covering automated software development workflows, custom actions development, and advanced pipeline configurations. You'll learn to create efficient CI/CD pipelines, implement security best practices, and optimize workflow performance. The curriculum combines theoretical knowledge with hands-on exercises, enabling you to automate software development processes effectively and streamline team collaboration through automated workflows.
- Courses:
    - GitHub Actions: The Big Picture
    - Authoring and Maintaining GitHub Actions Workflows
    - Consuming GitHub Actions Workflows
    - Authoring and Maintaining GitHub Actions
    - Managing GitHub Actions

## `Implementing DevOps with GitHub and Microsoft Azure` Skill Path

- Platform: Pluralsight
- Course Instructor: Multiple instructors
- Link: https://app.pluralsight.com/paths/skill/implementing-devops-with-github-and-microsoft-azure
- About: DevOps is one of those terms where any single person will give you a slightly different definition and way of doing it. Well, here’s another for you since this path explores DevOps methodologies using Github and Microsoft Azure. You’ll cover how to get your pipelines up and running using a combination of the vast capabilities offered by Azure and GitHub. As with all DevOps, we don’t intend for this to be a completely prescriptive way to do DevOps, but you’ll end up with a whole bag of new ideas to enact in your organization.
- Courses:
    - DevOps with GitHub and Azure: The Big Picture
    - DevOps with GitHub and Azure: Implementing Source Control with GitHub
    - DevOps with GitHub and Azure: Implementing CI/CD with GitHub Actions
    - DevOps with GitHub and Azure: Implementing Release Strategies with Azure
    - DevOps with GitHub and Azure: Implementing Package Management with GitHub
    - DevOps with GitHub and Azure: Implementing Software Supply Chain Security with GitHub
    - DevOps with GitHub and Azure: Implementing Infrastructure as Code
